Sobre o Projeto

Este site foi desenvolvido com Django e SQLite, com o objetivo de praticar e consolidar conhecimentos em desenvolvimento web full-stack.

Ele está atualmente hospedado na plataforma Render, permitindo fácil acesso e testes em ambiente de produção, sem a necessidade de configurar servidores locais.

Para o gerenciamento de imagens, está sendo utilizado o serviço da Cloudinary, garantindo maior performance no carregamento e manipulação das mídias.

Tanto os dados do carrinho de compras quanto dos produtos favoritos são armazenados via sessions, o que permite que as informações sejam mantidas mesmo sem a necessidade de login — facilitando a navegação e a experiência do usuário de forma anônima.

⚠️ O projeto ainda está em fase de desenvolvimento e pode apresentar pequenos bugs. Por exemplo: ao adicionar um produto aos favoritos, o usuário é redirecionado para a página inicial — comportamento que será ajustado futuramente.

A interface utiliza um CSS básico, pois o foco principal do projeto é a estrutura lógica e o funcionamento da aplicação. Ainda assim, sugestões de melhorias visuais são muito bem-vindas!

Se você tiver interesse em colaborar com ajustes de estilo ou corrigir alguns problemas, o repositório está disponível no GitHub: github.com/ronesanttos/virtual_store
