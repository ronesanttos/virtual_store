Sobre o Projeto

Este site foi desenvolvido utilizando Django e SQLite, com o objetivo de praticar e consolidar conhecimentos em desenvolvimento web.

Os dados do carrinho de compras e dos produtos favoritos são armazenados utilizando session, o que permite manter as informações mesmo sem login.

O projeto ainda está em desenvolvimento e pode conter alguns bugs. Por exemplo, ao adicionar um produto aos favoritos, o usuário é redirecionado para a página inicial — comportamento que ainda será ajustado.

O CSS utilizado é básico, pois a parte visual não é o foco principal deste projeto. No entanto, melhorias são bem-vindas!

Se você tiver interesse em colaborar com ajustes de estilo ou corrigir alguns problemas, o repositório está disponível no GitHub: github.com/ronesanttos/virtual_store
